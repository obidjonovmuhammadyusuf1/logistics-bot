# 🚚 Logistics Bot

**Yuk mashina haydovchilari uchun maxsus Telegram bot**

Prompt asosida yaratilgan: xarajatlar kuzatuvi, reys rejalashtirish, hisobotlar va boshqa yordam funksiyalari.

## Asosiy imkoniyatlar
- Xarajat qo'shish (solarka, ovqat va h.k.) + tasdiqlash
- Turli valyutalar (Rubl, Tenge, So'm) alohida hisoblanadi
- Bugun / Haftalik / Oylik hisobotlar
- Sheva va oddiy tilni tushunish
- Do'stona "katta aka" uslubi

## O'rnatish

1. BotFather orqali yangi bot yarating va token oling
2. `.env` fayl yarating:
   ```
   BOT_TOKEN=your_token_here
   ```
3. Kutubxonalarni o'rnating:
   ```bash
   pip install -r requirements.txt
   ```
4. Botni ishga tushiring:
   ```bash
   python bot.py
   ```
